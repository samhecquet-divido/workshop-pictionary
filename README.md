# Pictionary

Real-Time pictionary to test Socket!

I stole some code from [here](https://medium.com/swlh/build-a-real-time-chat-app-with-react-hooks-and-socket-io-4859c9afecb0) and [here](https://github.com/socketio/socket.io/tree/master/examples/whiteboard)

## Demo
https://user-images.githubusercontent.com/93543104/180490839-6f141b6f-8ce5-446b-911a-37061294449e.mov

## Installation

```bash
# ./client
yarn install
```

```bash
# ./server
yarn install
```

## Run

Client:

```bash
# ./client
yarn start
```

Server:

```bash
# ./server
node index.js
```
